Variable Hoisting

  • In JavaScript, a variable can be declared after it has been used. 
    
x = 5; // Assign 5 to x
document.getElementById("demo").innerHTML = x;
// Display the value of x in the element
var x; // Declare x
  • JavaScript only hoists declarations, not initializations.

Undefined and undeclared variable

  • Undefined Variables are those that are not assigned, but declared in the program.
  • Undeclared Variables are those that are not declared in the program.
  • If Undeclared Variables are assigned to some value, then implicit declaration is done.

var a;
alert(a); // a is undefined
alert(b); // b is undeclared

use strict

Defines that JavaScript code should be executed in "strict mode". With strict mode, you can not use undeclared variables. Strict mode makes it easier to write "secure" JavaScript.

this keyword

  • In JavaScript this always refers to the “owner” of the function we're executing, or to the object that a function is a method of.
  • In the global execution context, this refers to the global object, whether in strict mode or not. 
    
this.a = 37;
console.log(window.a); // 37
  • Inside a function, the value of this depends on how the function is invoked. 
    
function s() {
  console.log(this);
}

var a={ b: s };

s();   // refers to the window
a.b(); // refers to the object a

Inheritance

  • Javascript is a prototype-based language. Every object can then be used as a prototype for another object.
  • Objects created using an object literal, or with new Object(), inherit from a prototype called Object.prototype. Objects created with new Date() inherit the Date.prototype. The Object.prototype is on the top of the prototype chain.
  • Classical Inheritance involves organizing classes into a hierarchy, and enabling these classes to inherit attributes and behavior from classes above in the hierarchy.
    • 
function Person() {
    this.firstName = "Abc";
    this.lastName = "Xyz";
    this.age = 20;
}
//adding new property
Person.prototype.city = "kolkata"; 
//adding new methods
Person.prototype.name = function() {
    return this.firstName + " " + this.lastName;
};

var someone = new Person();
console.log(someone.city); // Kolkata
console.log(someone.name()); //Abc Xyz
